# Temperature Converter

A simple C program that converts between Celsius and Fahrenheit.

## Description
This program allows users to choose between two conversion options:
- Convert Fahrenheit to Celsius
- Convert Celsius to Fahrenheit

## How to Use
1. Run the program
2. Choose your conversion type:
   - Enter `1` for Fahrenheit to Celsius
   - Enter `2` for Celsius to Fahrenheit
3. Enter the temperature you want to convert
4. The program will display the converted result

## Example
```

from Fahrenheit to Celsius 1, Or from Celsius to fahrenheit enter 2: 1
Enter the Degree = 100

```
